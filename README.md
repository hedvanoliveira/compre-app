# compre-app
API DE COMPRA DE APLICATIVOS

## Como inicializar o ambiente do projeto
Executar o comando `docker compose up` na raiz do projeto. Será criado os containers: Seq, Redis, SqlServer e a API do projeto.

## Acessos
Acessar o swagger da API:
http://localhost:5010/swagger/index.html

Acessar o painel do Seq para visualização dos logs da API:
http://localhost:5341/
